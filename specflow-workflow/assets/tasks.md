# 任务模板

> 按变更类型选择对应模板。由 Tasks 阶段自动选型。

| 变更类型 | 模板文件 |
|---------|---------|
| 功能性（新功能 / 修改 / 移除 / Bug 修复） | `tasks-functional.md` |
| 非功能（性能 / 安全 / 重构 / 技术债 / 可观测性 / 可靠性） | `tasks-nonfunctional.md` |
| 基础设施（升级 / 迁移 / CI / 配置变更） | `tasks-infra.md` |
| 轻量（文档 / 测试补充 / 合规 / License） | `tasks-lightweight.md` |
