## 常见错误

（由团队填写：列出团队在输入验证上犯过的错误，例如：

- 只在前端做输入验证，后端未校验
- 用黑名单过滤危险字符，遗漏新型攻击向量
- 字符串拼接 SQL，导致注入漏洞
- 模板引擎未开启自动转义，输出用户输入导致 XSS
- 文件上传只检查扩展名，未检查真实文件类型
- 富文本编辑器未做 HTML 净化，存储型 XSS
- CSRF 防护只覆盖 POST 请求，遗漏 PUT / DELETE
- 文件上传后保留原始文件名，导致路径穿越
- 输入验证错误信息泄露系统内部结构（如暴露字段名）
）
