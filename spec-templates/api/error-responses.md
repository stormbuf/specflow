## 常见错误

（由团队填写：列出团队在错误响应上犯过的错误，例如：

- 错误响应格式不统一，不同接口返回不同结构
- 用 HTTP 200 返回错误信息，状态码与实际语义不符
- 错误码散落各处，没有统一注册，出现重复
- 系统错误堆栈直接返回给客户端，泄露内部信息
- 错误信息为技术术语，用户无法理解
- 字段校验错误只返回"参数错误"，不指出哪个字段有问题
- 错误码用数字且无文档，无人知道 10003 代表什么
- 业务错误与系统错误混在一起处理，导致 500 频发
- 批量操作部分失败时，整体返回 500 而非逐条返回结果
）
